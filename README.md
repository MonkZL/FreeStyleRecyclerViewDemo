使用方式：

        android:id="@+id/list"
        android:layout_width="match_parent"
        android:layout_height="0dp"
        android:layout_weight="1"
        app:defaultHeight="100dp"
        app:targetHeight="300dp" />
        
        defaultHeight是最小高度
        targetHeight是最大高度
